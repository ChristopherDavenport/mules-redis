mules-redis - A Cache Implementation Backed by Redis

Quick Start

To use mules-redis in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

libraryDependencies ++= Seq(
  "io.chrisdavenport" %% "mules-redis" % "<version>"
)
